## Pagos: cómo se interpretaron

Las celdas de pago de tu planilla se parsearon así:

| En tu planilla | Interpretación |
|----------------|----------------|
| `ibo 10/03` | Efectivo, fecha 10/03/2026 |
| `mp 07/04` | Mercado Pago, fecha 07/04/2026 |
| `bco 11/03` | Transferencia, fecha 11/03/2026 |
| `ibo 05/11 + cuota 10/03` | Dos pagos: efectivo 05/11/2026 + efectivo 10/03/2026 |
| `38k a favor bco 19/02` | Transferencia 19/02/2026, monto $38.000 (a favor) |
| `---` | No aplica (descartado) |
| Vacío | Sin pago |

**Pagos sin monto explícito:** se importan con `montoCobrado: 0`. Cuando definas los precios en la app, podrás recalcular. Para ahora alcanza con que la cuota figure como pagada.

1. **Importar el JSON** (ver instrucciones en INSTRUCCIONES_IMPORTACION.md)
2. **Cargar precios de cuotas** por curso desde la pestaña Cursos
3. **Cargar la matrícula** desde Configuración
4. **Cargar los horarios de cada curso** desde Cursos → Editar
5. **Asignar horario a cada alumno** desde Alumnos (en bulk podríamos hacer un script si querés)
6. **Completar datos faltantes** (DNI, celulares) de los alumnos huérfanos
7. **Revisar grupos familiares** y agregar manualmente los hermanos que no se hayan detectado por celular
